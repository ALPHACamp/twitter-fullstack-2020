# # Simple Twitter (fullstack)

### 以Twitter為原型的社交平台

- 使用者須註冊帳號才能使用本網站功能
- 使用者可以張貼文章
- 使用者可以回覆文章
- 使用者可以追蹤,取消追蹤特定使用者
- 使用者可以編輯個人資訊
- 使用者可以查看自己或他人的資訊(包含貼文,回覆留言,追蹤者清單,喜歡的內容)
