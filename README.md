# Simple Twitter
## Features
- 除了註冊和登入頁，使用者一定要登入才能使用網站
- 使用者可以創建帳號、登入、登出
- 使用者可以編輯自己的賬號設定
- 使用者可以瀏覽/回復/新增推文
- 使用者可以跟隨/被跟隨
- 使用者可以like/unlike推文和留言
- 管理員可以登入後台
- 管理員可以瀏覽全站的用戶
- 管理員可以瀏覽全站的推文
- 管理員可以刪除使用者的推文
